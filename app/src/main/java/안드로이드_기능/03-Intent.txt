Intent
- 번역시, 의도 의사
- A야 B좀 해줘
- B좀 해줘
- A야 B좀 해줘 그리고 다하면 알려줘
- 데이터를 같이 전달할 수 있다 (어디에 전화를 걸어줘, 어떤 txt를 보내줘. 한 액티비티에서 다음 액티비티로 이 데이터들을 넘겨)

- Intent의 종류
    - 명시적 인텐트 (Explicit Intent)
        - 호출될 대상을 명시하는 경우
    - 암시적 인텐트 (Implicit Intent)
        - 호출될 대상을 명시하지 않는 경우
        - 인텐트 필터 (Intent- filter)
            - 모든 앱이 호출되면 안되기 때문에 존재
            - 암시적 인텐트를 보낸 경우, 인텐트가 처리할 수 있는지 확인할 때 사용
            - AndroidManifest 확인시 <intent-filter> 태그가 존재

- 인텐트 호출 대상 V
    - 앱내에서
        - 엑티비티 끼리
    - 외부(안드로이드 OS, 시스템), 앱 끼리도 사용 가능 (권한이 필요한다)
    - 사진 앱
        - 우리 앱 -> 시스템 -> 사진첩
        - 우리 앱 전화 걸기 버튼 -> 시스템 -> 전화 걸기 앱

- 인텐트 FLAG
    - 한 액티비티에서 다른 액티비티로 Intent로 전달하고 다시 예전 액티비티에서 결과가 필요할 수 있다.
    - 결과가 필요한 경우 startActivityForResult()
    - 결괄가 필요하지 않은 경우 startActivity
