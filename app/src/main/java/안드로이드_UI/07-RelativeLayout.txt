RelativeLayout
- 부모 뷰가 될 수 있는 컨테이너 뷰이다
- 리니어 레이아웃은 자식뷰의 배치를 가로, 세로로 설정
- 자식뷰의 배치를 설정 할 수 있다
    - 기준점을 중심으로 자식뷰를 배치
        - 기준점 : 부모, 자식뷰
        - 부모, 자식 뷰를 기준으로 잡기 위해서는 해당 부모, 자식 뷰를 식별해야 함.
    - 기준점을 표시하는 방법
        - id속성을 사용
    A B C D
    B : A를 기준으로 A의 오른쪽
    C : D를 기준으로 D의 왼쪽
    A는 B의 왼쪽 B는 C의 왼쪽 C는 D의 왼쪽 -> 이런식으로 할 시 문제가 발생 (동작은 할 수 있으나, 원래 원하던 형태로 생성되지 않을 수 있음)
